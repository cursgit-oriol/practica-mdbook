# Pràctica de mdBook

## Instruccions

L'objectiu d'aquesta pràctica és crear un llibre digital utilitzant **mdBook**.

### Passos a seguir:

1. **Instal·lar mdBook**:
   Descarrega l'executable de mdBook.

2. **Crear l'estructura**:
   Obre el terminal en aquesta carpeta i crea l'estructura amb la comanda:   `mdbook init el-meu-llibre`

3. **Editar el contingut**:
   Entra a la carpeta `el-meu-llibre-digital` i edita els fitxers Markdown dins de `src/` per afegir capitols al teu llibre.

4. **Publicar**:
   Configura GitHub Pages per mostrar el teu llibre.

## Lliurament

Només cal fer commits i push al teu repositori generat per GitHub Classroom. El professor revisarà el codi i el desplegament que has fet a GitHub Pages.
